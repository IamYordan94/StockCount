# Initial Data Setup Guide

## ✅ Current Status
- ✅ Database tables created
- ✅ Default categories exist (Food, Drinks, Cheese, Other)
- ✅ You're set as manager
- ⏳ Now you need to add your data!

## Step-by-Step Setup

### Step 1: Add Your Shops (11 locations)

1. In your app, go to: **Dashboard → Shops**
2. Click **"Add Shop"**
3. Add all 11 of your shop locations, one by one
   - Example: "Shop 1", "Shop 2", "Downtown Location", etc.
4. Repeat until all 11 shops are added

### Step 2: Add Your Products/Items

1. Go to: **Dashboard → Products**
2. Click **"Add Product"**
3. For each item, fill in:
   - **Item Name**: e.g., "Coca Cola 500ml"
   - **Pack Size**: e.g., "12 per pack" or "500g" or "24 bottles"
   - **Category**: Select from Food, Drinks, Cheese, or Other
4. Click **"Create"**
5. Repeat for all your products

**Tip**: You mentioned you have a full list of items. You can add them all now, or add them as you go.

### Step 3: Assign Items to Shops

1. Go to: **Dashboard → Products**
2. Click **"Assign Items to Shops"** button
3. Select a shop from the dropdown
4. Check the boxes next to items that should appear in that shop
5. Repeat for each shop

**Note**: 
- If you don't assign items to a shop, ALL items will show for that shop
- If you assign specific items, only those items will show

### Step 4: Add Users (Your Employees)

1. Go to: **Dashboard → Users**
2. Click **"Add User"**
3. Fill in:
   - **Email**: employee@example.com
   - **Password**: (they can change it later)
   - **Role**: Employee
   - **Assigned Shops**: Check which shops this employee should count
4. Click **"Create"**
5. Repeat for all employees

### Step 5: Create Your First Stock Count Session

1. Go to: **Dashboard → Stock Count Sessions**
2. Click **"Create New Session"**
3. Name it (e.g., "January 2024" or "December Stock Count")
4. Click **"Create"**

Now employees can:
- Go to Dashboard → Count Stock
- Select the session
- Select their assigned shop
- Start counting!

## Tips

- **Bulk Adding**: You can add shops and products quickly one after another
- **Shop Items**: If most shops have the same items, you can skip assigning and let all items show for all shops
- **Categories**: The 4 default categories should cover most items. If you need more, we can add them later.

## Ready to Start?

Go to your app at http://localhost:3000 and start with **Dashboard → Shops**!
